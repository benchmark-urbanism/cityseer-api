% Conclusion section

We have presented an adaptive per-distance sampling approach for efficient multi-scale network centrality computation. The central insight---that effective sample size is a sufficient statistic for predicting accuracy---enables per-distance calibration of sampling probability, achieving consistent accuracy across all distance thresholds while reducing computation time substantially at longer distances.

Adaptive sampling addresses a practical bottleneck in urban network analysis. At regional scales (5--20km distances), metropolitan networks may contain 100,000+ nodes where full computation requires hours; adaptive sampling reduces this to minutes while preserving ranking accuracy. The speedup enables more rapid iteration for scenario testing and makes real-time analysis feasible for planning support tools.

A key enabling factor is the use of \emph{target aggregation} in the centrality computation, where values are accumulated at reachable target nodes rather than at source nodes. This ensures that even nodes not selected as sample sources receive contributions from sources that reach them, providing high coverage and spatial smoothing of sampling error.

As cities increasingly use network analysis to inform planning decisions---from pedestrian accessibility studies to metropolitan transit planning---efficient and accurate methods become essential. Adaptive per-distance sampling provides a principled approach to balancing computational cost against accuracy requirements, making large-scale multi-modal network analysis practical for routine use.
