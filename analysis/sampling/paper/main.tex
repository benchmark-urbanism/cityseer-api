\begin{abstract}
  We show that \emph{effective sample size}---the product of mean network reachability and sampling probability---is a sufficient statistic for predicting centrality estimation accuracy across network topologies and distance thresholds. This insight enables adaptive per-distance sampling that maintains consistent accuracy where uniform sampling fails. Through experiments on synthetic networks, we fit empirical models predicting Spearman rank correlation from effective sample size, finding that harmonic closeness and betweenness centrality have distinct accuracy characteristics that vary by distance heuristic. Our adaptive algorithm uses these models to automatically calibrate sampling probability for each distance threshold; as distance increases, so too does the speedup, while accuracy remains consistent at $\rho \geq 0.95$ across all scales. Validation on real-world street networks from London, Madrid, and Phoenix confirms that the empirical models generalise beyond synthetic test cases. The implementation is available in the open-source \cityseer{} Python library.
\end{abstract}

\textbf{Keywords:} network centrality, sampling, urban morphology, betweenness, closeness, computational efficiency
